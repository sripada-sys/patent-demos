# Patent Demos — Student Guide

Each folder/file in `demos/` is a working implementation of an expired patent.
Open the HTML file in your browser (or phone) and it runs. No install. No build step.

## How to use these demos

1. Read the patent summary in `docs/patents/`
2. Open the demo in your browser
3. Run it on your phone — the sensors are all built in
4. Read the source — it's short by design

## What you'll need

**Browser demos:** Any modern browser. Best on a phone so you get real GPS.

**Hardware demos:** ~$20 in components. Exact parts listed per episode.

## Why expired patents?

Patents expire 20 years after filing. After expiry, anyone can use, implement, and teach from them freely.
The patents here were foundational — they shaped an entire industry. Now they're yours to learn from.
